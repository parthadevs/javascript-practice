# 🚀 JavaScript Daily Practice

A personal repository where I practice JavaScript every day to strengthen my fundamentals and gain a deeper understanding of how JavaScript works under the hood.

Although I've built several real-world projects using modern frameworks and AI-assisted development, I realized that there are still gaps in my JavaScript fundamentals. This repository is my commitment to fixing those gaps by going back to the basics and learning JavaScript the right way.

My goal is not just to write JavaScript, but to understand why it works

---

## 📌 My Goals
* Build a strong foundation in Core JavaScript.
* Master JavaScript fundamentals before relying on frameworks.
* Deep dive into concepts like Closures, Scope, Execution Context,     Prototypes, Event Loop, and Asynchronous JavaScript.
* Understand how frameworks like Next.js and NestJS work under the hood.
* Solve coding challenges and practice problem solving every day.
* Maintain consistency by committing code daily.
